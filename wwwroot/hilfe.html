Verfügbarkeit eintragen

  1. Auf einen Tag im Kalender tippen
  2. Es erscheint ein Auswahl-Dialog: Frei / Vielleicht / Keine Zeit
  3. Optional eine Notiz hinzufügen
  4. Mit Speichern bestätigen
💡 Grüne Tage (mit Haken-Symbol) bedeuten: alle Mitglieder sind an dem Tag frei – ideal für Proben oder Konzerte.

Termine anlegen (nur Admins)

  1. Auf einen Termin-Tag tippen (pinkes Ausrufezeichen am Datum)
  2. Im Dialog auf + Termin klicken
  3. Name, Datum und optional Uhrzeit eingeben
  4. Speichern – der Termin erscheint im Kalender und im ICS-Feed

Song hinzufügen

  1. Oben rechts auf + Song tippen
  2. Titel eingeben
  3. Kategorie wählen: Eigener Song · Cover · In Arbeit · Idee
  4. Optional eine Notiz hinzufügen
  5. Mit Hinzufügen bestätigen

Aufnahme hochladen

  1. Song-Karte aufklappen (antippen)
  2. Im Bereich Aufnahmen auf ↑ Hochladen tippen
  3. Audio- oder Videodatei auswählen (MP3, M4A, WAV, MP4 …)
  4. Die Datei erscheint mit Dauer und Dateigröße in der Liste
🎙 Direkt aufnehmen: Auf 🎙 Aufnehmen tippen, um das Mikrofon des Geräts zu nutzen. Nach der Aufnahme kann sie angehört und hochgeladen oder verworfen werden.

Song bewerten

In jeder aufgeklappten Song-Karte gibt es Stern-Buttons (1–5). Einfach die gewünschte Bewertung tippen und optional einen Kommentar schreiben, dann Speichern.

Die Durchschnittsbewertung aller Mitglieder wird im Karten-Header angezeigt.

Aufnahmen bewerten

Jede einzelne Aufnahme kann ebenfalls bewertet werden. Das ist wichtig für das Setlisten-Abspielen – dort wird automatisch die am besten bewertete Aufnahme pro Song gewählt.

Link zu einer Aufnahme teilen (🔗)

  1. Song aufklappen → bei der gewünschten Aufnahme auf 🔗 tippen
  2. Der Link wird in die Zwischenablage kopiert
  3. Link z.B. via WhatsApp teilen
  4. Wer auf den Link klickt, landet direkt bei der Song-Karte (wenn die App installiert ist, öffnet sie direkt in der App)

Mitglieder per Push benachrichtigen (🔔)

  1. Bei einer Aufnahme auf 🔔 tippen
  2. Alle anderen Mitglieder erhalten eine Push-Nachricht mit dem Hinweis, die Aufnahme anzuhören und zu bewerten
  3. Ein Klick auf die Benachrichtigung öffnet direkt die richtige Song-Karte

📋 Setlisten

Erstellt und verwaltet Setlisten für Konzerte oder Proben. Songs können per Drag & Drop sortiert werden. Die Setliste kann als PDF exportiert und als Playlist abgespielt werden.

Konzert Mühldorf ✏ ▶ ⬇ ⧉ ×
08.08.2026 · 9 Songs · ⏱ ~42 Min · ★★★★☆

Setliste erstellen

  1. Auf + Setliste tippen
  2. Name eingeben (z.B. "Konzert Mühldorf")
  3. Optional: Datum und Notizen hinzufügen
  4. Mit Erstellen bestätigen

Songs hinzufügen

  1. Setliste aufklappen
  2. Unten im Bereich Songs auf + Song hinzufügen tippen
  3. Song aus der Liste auswählen
  4. Reihenfolge per Drag & Drop (☰-Symbol) anpassen
🪄 Autofill: Die Setliste kann automatisch mit den am besten bewerteten Songs befüllt werden – entweder nach Anzahl oder nach Gesamtdauer. Einfach im aufgeklappten Bereich auf Autofill tippen.

Setliste abspielen (▶)

  1. Im Setlisten-Header auf ▶ tippen
  2. Ein Player öffnet sich von unten
  3. Pro Song wird automatisch die am besten bewertete Aufnahme gewählt
  4. Songs ohne Aufnahme werden übersprungen
  5. Mit ⏮ / ⏭ zwischen Songs wechseln
  6. Am Ende eines Songs wird automatisch der nächste gestartet

Als PDF exportieren (⬇)

  1. Im Setlisten-Header auf ⬇ tippen
  2. Ein PDF wird direkt im Browser generiert und kann gespeichert oder gedruckt werden

Setliste kopieren (⧉)

Erstellt eine exakte Kopie der Setliste mit dem Zusatz "(Kopie)" – nützlich um Varianten zu erstellen.

Aufgabe erstellen

  1. Oben rechts auf + Aufgabe tippen
  2. Titel und optionale Beschreibung eingeben
  3. Optional Fälligkeitsdatum setzen
  4. Mit Erstellen bestätigen

Aufgabe bearbeiten

  1. Auf eine Aufgaben-Karte tippen – ein Detailfenster öffnet sich
  2. Hier können Mitglieder zugewiesen, Checklisten hinzugefügt und die Spalte gewechselt werden
  3. Aufgaben können per Drag & Drop zwischen den Spalten verschoben werden

Checkliste hinzufügen

  1. Aufgabe öffnen → + Punkt hinzufügen im Checklisten-Bereich
  2. Punkte können abgehakt werden – der Fortschritt wird in der Karten-Vorschau angezeigt

Neues Finanz-Event anlegen (nur Admins)

  1. Auf + Event tippen
  2. Name (z.B. "Konzert Rosenheim"), Datum und optionale Beschreibung eingeben
  3. Bandkasse-Anteil (%) festlegen

Ausgabe erfassen

  1. Finanz-Event aufklappen
  2. Auf + Ausgabe tippen
  3. Betrag und Beschreibung eingeben
  4. Optional: Kassenbon-Foto hochladen
  5. Mit Speichern bestätigen

Einnahme erfassen

  1. Finanz-Event aufklappen
  2. Auf + Einnahme tippen
  3. Betrag und optionale Beschreibung eingeben
  4. Mit Speichern bestätigen

👤 Account & Admin

Jedes Mitglied hat einen eigenen Account mit Farbe und Name. Admins können weitere Mitglieder anlegen und verwalten.

Mein Account

  • Name: Bandname (für alle sichtbar)
  • Echter Name: Nur für dich selbst sichtbar – du siehst deinen eigenen Namen, andere sehen deinen Bandnamen
  • Farbe: Persönliche Farbe zur Kennzeichnung im Kalender und überall in der App

Mitglied anlegen (nur Admins)

  1. Im Account-Bereich auf + Mitglied tippen
  2. Name und Passwort eingeben, Farbe wählen
  3. Mit Hinzufügen bestätigen
  4. Das neue Mitglied kann sich sofort einloggen

🔔 Benachrichtigungen

Die App kann Push-Benachrichtigungen schicken – z.B. wenn eine neue Aufnahme hochgeladen wurde. Benachrichtigungen funktionieren auch wenn die App gerade nicht offen ist.

Benachrichtigungen aktivieren

  1. Beim ersten Besuch erscheint ein pinker Banner am unteren Bildschirmrand
  2. Auf Aktivieren tippen und die Browser-Anfrage bestätigen
  3. Alternativ: In der App auf das 🔔-Symbol in der Kopfzeile tippen
📱 Als App installieren (Android): In Chrome auf ⋮ → App installieren tippen. Danach erscheint die App auf dem Homescreen und öffnet sich ohne Browser-Rahmen.
📱 Als App installieren (iPhone): In Safari auf Teilen → Zum Home-Bildschirm tippen. Benachrichtigungen funktionieren auf iOS nur wenn die App über Safari auf iOS 16.4+ installiert wurde.

Wann kommen Benachrichtigungen?

  • Wenn jemand eine Aufnahme hochlädt und auf 🔔 tippt
  • Die Benachrichtigung enthält Songname und Uploader
  • Klick auf die Benachrichtigung → App öffnet direkt die Song-Karte
